Streamlit Visualization

Create interactive dashboards with Streamlit. Place your Python files in the visualize/ folder of your workspace.

Quick Start

  1. Open the Notebooks tab
  2. Create a folder named visualize in your workspace root
  3. Create a Python file (e.g., visualize/dashboard.py) with Streamlit code
  4. Return to this tab to see your visualization

Example: visualize/dashboard.py

import streamlit as st
import pandas as pd

st.title("My Dashboard")

data = pd.DataFrame({
    "x": range(10),
    "y": [i**2 for i in range(10)]
})
st.line_chart(data.set_index("x"))
